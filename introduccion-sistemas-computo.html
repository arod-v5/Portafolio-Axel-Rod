Mantenimiento preventivo


Mantenimiento basico y limpiaza a una PC de estritorio

Descripción

Para el desarrollo del primer proyecto se organizaron equipos de trabajo con el objetivo de facilitar la colaboración entre compañeros, apoyarse mutuamente y reducir costos en materiales, en caso de que algún integrante no contara con los materiales o herramientas necesarias. El proyecto consistía en realizar mantenimiento preventivo a una computadora, aplicando los conocimientos básicos relacionados con hardware y limpieza de equipos.


Entre los materiales utilizados se encontraban espuma limpiadora, limpia contactos, aire comprimido, un poco de wipe y otros elementos. Herramientas importantes para limpiar correctamente los componentes sin dañarlos. Posteriormente, procedimos a desarmar cuidadosamente una computadora para identificar cada una de sus partes internas, como la tarjeta madre, memoria RAM, disco duro, fuente de poder y ventiladores.


Después de identificar los componentes, realizamos el proceso de limpieza utilizando los materiales correspondientes para eliminar polvo y suciedad acumulada que pudiera afectar el rendimiento del equipo. Finalmente, se volvió a ensamblar la computadora y se elaboró un reporte técnico documentando el procedimiento realizado, los componentes identificados y se realizó una investigación de los componentes de la motherboard.

Instalacion Ubuntu


Instalar Ubuntu en Virtual Box y probar el CMD


Descripción


El segundo proyecto se enfocó en el uso de máquinas virtuales y sistemas operativos. Inicialmente, tuvimos que instalar Virtual Box en nuestras computadoras para poder ejecutar un segundo sistema operativo de manera simulada dentro del equipo principal.


Antes de instalar la máquina virtual, fue necesario ingresar a la BIOS de la computadora y habilitar la opción de virtualización, ya que sin esta configuración el programa no podía funcionar correctamente. Luego descargamos la ISO del sistema operativo Ubuntu y procedimos a instalarlo dentro de la máquina virtual configurando memoria RAM, almacenamiento y otros parámetros necesarios.


Una vez instalado el sistema operativo, verificamos su funcionamiento y exploramos sus características básicas a su vez probamos comandos dentro de la consola de Ubuntu como: sudo su, echo, cat, etc.

Cables de Red


Creacion de dos tipos de cable tipo A - A y A - B


Descripción


El tercer proyecto combinó teoría y práctica relacionada con cableado estructurado y las redes. Inicialmente aprendimos las configuraciones estándar para cables Ethernet, específicamente la configuración tipo A y tipo B.


Utilizando cable UTP, conectores RJ45 y un kit de ponchado, procedimos a fabricar nuestros propios cables de red. Para la fabricación usamos 6 metros de cable CAT 5 partiéndolo a la mitad conseguimos dos cables, para el primer cable realizamos un cable directo utilizando configuración B en ambos extremos (B-B), mientras que el segundo fue un cable cruzado utilizando configuración A en un extremo y configuración B en el otro (A-B).


Posteriormente, utilizamos un probador de conectividad incluido en el kit de ponchado para verificar que los cables estuvieran correctamente armados y funcionando adecuadamente, para esto tenían que hacer un recorrido limpio entre la luz 1 y la 8.

Configuracion de una red LAN


Creacion red LAN mediante los cables previamente creados


Descripción


El cuarto proyecto está estrechamente relacionado con el anterior proyecto, ya que utilizamos los cables de red fabricados para crear una red LAN (Local Area Network) entre varias computadoras. Para esto utilizamos un switch de red que permitió interconectar todos los equipos dentro de la misma red local.


Posteriormente, realizamos configuraciones desde el CMD asignando direcciones IP manualmente a cada computadora para permitir la comunicación interna entre los dispositivos, realizamos diversos ping a las computadoras conectadas y salieron exitosos.


Posteriormente, debíamos de compartir archivos entre computadoras y anqué no salieron todos de manera exitosa, sí conseguimos establecer conexión entre algunos equipos, ya para finalizar contestamos algunas preguntas planteadas.

Red LAN -- Packet Tracer


Creacion de una red LAN en cisco packet tracer utilizando un modelo de una casa de tres niveles


Descripción


Este proyecto consistió en realizar una simulación de redes utilizando Cisco Packet Tracer. Básicamente, aplicamos los conocimientos aprendidos en el proyecto anterior, pero esta vez dentro de un entorno virtual de simulación.


El objetivo era diseñar la red estructurada en una casa de tres niveles, conectando múltiples computadoras, laptops y diferentes dispositivos de red. Cada piso debía trabajar con diferentes segmentos IP y debíamos lograr organizar correctamente la comunicación entre dispositivos.


Después de realizar todas las conexiones y configuraciones correspondientes, verificamos el funcionamiento de la red enviando paquetes de datos entre las computadoras simuladas dentro del programa.

Base de datos


Solucion de tres tablas de diferentes ejercicios


Descripción


Para el sexto proyecto trabajamos con bases de datos utilizando dos aplicaciones principales: XAMPP y MySQL Workbench.


Inicialmente, utilizamos Workbench para diseñar el modelo de varias bases de datos, creando tablas y estableciendo relaciones entre ellas. Después de finalizar el diseño, generamos el script SQL correspondiente que vendría siendo el código de la tabla.


Posteriormente, mediante XAMPP accedimos desde el navegador a phpMyAdmin, donde pegamos y ejecutamos el script generado previamente para crear la base de datos dentro del servidor local. Finalmente, ingresamos datos en las tablas relacionadas para comprobar que la base de datos funcionara correctamente y que las relaciones entre tablas estuvieran bien estructuradas.
