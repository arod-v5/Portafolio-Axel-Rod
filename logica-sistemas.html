Creacion de un circuito


Resolver el ejercicio 3 de la hoja de trabajo propuesta

Descripción

Se llevará a cabo un proceso completo de análisis, diseño e implementación de un circuito digital combinacional para un tanque de agua en un residencial con dos bombas de salida. El objetivo principal será construir un sistema lógico capaz de activar o desactivar ambas bombas de agua.


Inicialmente, se identificarán las variables de entrada y salida del sistema. Como entradas se utilizarán los sensores de nivel de agua “A”, “B”, “C” y “D”. Las salidas serán las bombas M1 y M2. Posteriormente, se analizará cuidadosamente el funcionamiento solicitado para comprender en qué situaciones debe activarse cada bomba y cuándo deben apagarse.


Después del análisis, se procederá a construir la tabla de verdad del sistema. En esta tabla se representarán todas las combinaciones posibles de los sensores y el comportamiento correspondiente de las bombas. Esto permitirá visualizar de forma lógica cada caso planteado, por ejemplo: cuando el agua esté por debajo del sensor “C”, cuando el nivel esté entre “A” y “C”, o cuando el tanque esté lleno.


Una vez terminada la tabla de verdad, se obtendrán las funciones booleanas para las salidas M1 y M2. Estas funciones representarán matemáticamente el comportamiento lógico del circuito. Luego, se simplificarán utilizando Álgebra de Boole, con el objetivo de reducir la cantidad de compuertas necesarias y hacer el circuito más eficiente.


Con las funciones simplificadas, se diseñará el diagrama lógico del circuito utilizando compuertas AND, OR y NOT. En esta etapa se representará gráficamente cómo se conectarán los sensores y las compuertas para producir las señales de salida correctas hacia las bombas M1 y M2.


Posteriormente, se realizará la simulación del circuito en un software de diseño digital para verificar que el sistema funcione correctamente en todos los casos posibles. Después de comprobar el funcionamiento lógico, se implementará físicamente el circuito utilizando compuertas lógicas reales, cables de conexión, protoboard y componentes electrónicos necesarios.


Finalmente, se documentará todo el proceso en un informe formal con normas APA, incluyendo introducción, objetivos, marco teórico, planteamiento del problema, desarrollo metodológico, tabla de verdad, simplificación de funciones booleanas, diagramas lógicos, implementación física, resultados obtenidos y conclusiones del proyecto.
